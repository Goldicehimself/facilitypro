# FacilityPro

FacilityPro is a multi-tenant SaaS platform designed to help organizations manage facility maintenance operations efficiently. The system allows companies to track assets, create and assign work orders, manage vendors, and monitor maintenance activities through a centralized dashboard.

---

## 🌐 Live Application

[Visit FacilityPro](https://facilitypro-eight.vercel.app)

---

## ⚡ Features

- Multi-tenant architecture supporting multiple organizations  
- Work order creation and task tracking  
- Vendor management system  
- Asset tracking and maintenance history  
- Role-based access control (Admin, Manager, Technician)  
- Secure authentication and authorization  
- Image upload support for assets and user profiles  
- Responsive user interface

## 📂 System Capabilities

FacilityPro enables facility managers to:  

- Create and manage maintenance work orders  
- Assign technicians to maintenance tasks  
- Track equipment and asset maintenance history  
- Manage vendors and service providers  
- Monitor operational maintenance workflows  

---

## 🏗 Architecture

The platform uses a **multi-tenant architecture**, allowing multiple organizations to operate independently within the same system while keeping their data securely isolated.
